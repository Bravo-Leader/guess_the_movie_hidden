Politique de Confidentialité

Dernière mise à jour : 13 avril 2025

Extension GuessTheMovie Cleaner

Collecte de Données

Aucune donnée utilisateur n'est collectée par cette extension.

Cette extension fonctionne entièrement localement dans votre navigateur et ne :

  • Ne collecte aucune information personnelle
  • Ne stocke aucune donnée
  • Ne communique avec aucun serveur externe
  • Ne surveille pas votre activité en ligne
  • Ne partage aucune information avec des tiers

Fonctionnement de l'Extension

L'extension GuessTheMovie Cleaner :

  • Masque le numéro de jeu et la date sur le site GuessTheMovie.Name
  • Nettoie l'URL pour supprimer le numéro de jeu
  • Modifie le titre de l'onglet pour masquer les informations de jeu
  • Fonctionne uniquement sur le site https://guessthemovie.name/p/*

Autorisations Requises

L'extension demande les autorisations suivantes :

  • Scripting : Nécessaire pour modifier le contenu de la page web
  • Accès au site : Limité uniquement à https://guessthemovie.name/p/*

Contact

Si vous avez des questions concernant cette politique de confidentialité, vous pouvez nous contacter via le Chrome Web Store.

Modifications

Cette politique de confidentialité peut être mise à jour. Les modifications seront publiées sur cette page avec une nouvelle date de mise à jour.
